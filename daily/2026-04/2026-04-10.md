## 📌 话题四：Copilot Portal 反代 + Multi-Agent 架构 🔥

**发起人：** He Zhang  
**时间：** 4/9 全天（最活跃话题）

### A. Copilot Portal — 用 GHC quota 免费跑 Opus
- He Zhang 开源了 [copilot-portal](https://github.com/GhostComplex/copilot-portal)：Azure 上反代，用 GHC auth token 中转 LLM 请求
- 效果：API mock 成 Anthropic/OpenAI 兼容格式，不用翻墙也能用 Opus
- 绑定个人 GitHub 账号也能用，key 在公司电脑拿一次就行

### B. 自我迭代的 Agent — isotopes 项目
- He Zhang 分享了 isotopes 项目的第一个 self-generated PR：[isotopes/pull/59](https://github.com/GhostComplex/isotopes/pull/59)
- Agent 自己提交了代码改动，实现了 "channel 里 bot 之间互相 @可见" 的功能

### C. Multi-Agent 配置与权限管理
- 主力机跑 7 个 agent，另外 2 台 VM 各 2 个 = 总共 11 个
- 核心挑战：多个 agent 在同一环境里怎么管理谁能看到什么、谁该回复
- 解决方案：openclaw 配置 + Discord role 管理 + sac-spawn skill

**🧠 解读 & 应用：**
Copilot Portal 可以在非公司设备免费用 Opus。Multi-Agent 是多个 AI agent 协作分工的前沿实践。Self-evolving agent 是 AGI 方向的早期实验。

`#copilot-portal` `#multi-agent` `#self-evolving` `#openclaw`
