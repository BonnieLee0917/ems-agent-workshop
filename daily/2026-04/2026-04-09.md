## 📌 话题一：Copilot Portal — GHC 额度反代方案

**发起人:** He Zhang
**参与者:** Xiaolin Quan, Mike Li, Zheng Li

**内容:**

He Zhang 开源了 [copilot-portal](https://github.com/GhostComplex/copilot-portal)：在 Azure 上搭了一个反代，用 GHC（GitHub Copilot）的 auth token 中转 LLM 请求。

- 用法：`npx copilot-portal auth` 获取 token，endpoint 指向 Azure webapp
- API 格式 mock 成 Anthropic/OpenAI 兼容，所有工具（Claude Code、openclaw 等）都能通
- 好处：① 用公司 GHC enterprise quota（目前看似无限）② 不用翻墙也能用 Opus ③ 在非 enroll 设备也能用
- Key 在公司电脑拿一次就行，不变的，到处能用
- 绑定个人 GitHub 账号也能用（去 https://aka.ms/copilot 绑定）
- Mike Li 确认：都是 enterprise 的
- He Zhang 解释为什么放 Azure：Anthropic 封 data center IP 比较激进，个人代理不稳定，Azure 对境内友好
- Cloudflare workers.dev 需要翻墙所以被放弃了

> 📷 He Zhang 发了 copilot-portal 使用截图，展示了 endpoint 配置方式

**🧠 解读 & 应用建议:**
如果想在非公司设备上也用 Opus 模型，这是目前最方便的方案。本质是利用 GHC 的 enterprise 额度做 API 中转。
