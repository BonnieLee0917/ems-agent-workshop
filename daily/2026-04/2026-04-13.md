## 🎒 话题一：Kids Vibe Coding 与 2K 赛道

**发起人**：Jingxia Xing, Xiaolin Quan, Zheng Li, Luna Chen | **时间**：07:44 - 10:17

Jingxia 分享了女儿做的 app 截图，引发了一场关于"下一代 vibe coder"的热烈讨论。

**核心对话**：

- **Jingxia** 分享女儿的 app（📷 请到群里看截图），Dazhen Pan 感叹："完了，三天不学习，赶不上 Sirui 刑"
- **Jingxia** 解释女儿的需求：周围朋友喜欢找她吐槽，吐槽多了有负能量，所以做了一个**树洞 app**。还顺带做了 mentor-table 功能，快要上线了
- **Jingxia** 自嘲："话说有要考国际数学竞赛的小窗我，我可以卖 app，做的老完备了，就是 DAU == 0"。女儿不用爹做的，自己做了一个
- **Xiaolin Quan**："我的娃也在 vibe coding，现在上瘾和 Minecraft 不相上下，昨天说想做个 Minecraft 出来..."（📷 截图）
- **Zheng Li** 提出新概念："第三个赛道出现了，不只有 2B、2C，看来还有 **2K（Kids）**"
- **Luna Chen**："kids 更了解 kids，kids 才能做出 kids 要的东西"
- **Zheng Li** 建议："引导他们先做个小的，慢慢迭代，避免一开始憋个大的做不出来，打击积极性"

🧠 **解读**：Jingxia 女儿的例子完美说明了一个产品道理：**最好的产品来自真实需求，而不是想象的需求**。她爹（资深工程师）做了数学竞赛 app，DAU=0。女儿自己做了树洞 app，同学都在用。这就是 product-market fit 的活教材。Zheng Li 的"2K 赛道"不是段子。AI 降低了编程门槛后，青少年可以直接做产品给同龄人用。

## ⚠️ 话题二：He Zhang 的 GitHub 账号被封

**发起人**：He Zhang, Zheng Li | **时间**：09:05 - 09:17

He Zhang 的 GitHub 账号被封了。背景：他在 Azure VM 上跑 agent，自动管理 repo（开 issue、提 PR、review PR），已经限制了 agent 只能操作自己的 repo。

- **He Zhang**："应该也没有啥特别离谱的操作，org 和 repo 还在，号没了"
- **He Zhang**："我怕它在别人 repo 拉屎，还特别限制了它只能在我这个 repo 搞 😂"
- **Zheng Li**："你干啥了，是不是 IP 特别杂"→"可能 Azure 机房 IP 比较容易被识别"
- **He Zhang**："我再等两天看看工单咋回。估计对面还在周末"

🧠 **解读**：agent 自动化的真实风险案例。GitHub 对异常行为的检测越来越严格，Azure VM 的 IP 容易被识别为自动化行为。警示：避免使用云服务器 IP 跑高频 GitHub 操作，控制 API 调用频率，确保有备用账号。

#github-ban #agent-risk #automation-safety #azure-vm

## 🦞 话题三：龙虾（OpenClaw）Harness 方法论

**发起人**：Miaomiao Lei, Jingxia Xing, He Zhang | **时间**：11:03 - 11:26

- **Miaomiao Lei** 吐槽："我的龙虾经过两个星期的调配，在跟我一起做完一个完整项目之后，还是像个大傻子……远不如我的 PM Studio 来得可控"
- **Jingxia Xing** 点出核心："**PM Studio 替你做了 harness，龙虾需要你自己做 harness**。你可以找一个成型的方法论帮你 harness 龙虾"
- **He Zhang**："等我 isotope 迭代完也许行，有一个功能就是冷热切换，让它自己修复自己。longxia 我已经放弃让它自己改 config 了，经常把自己改坏了"
- **Jie Tang**："把 Claude Code 代码丢给龙虾让它自己学，能学会吗？"
- **Jingxia**："没那么容易。他们的架构不一样，严格来说很不一样。龙虾能出圈因为和 CC 不是一个路数"

🧠 **解读**：**工具好不好用，取决于 harness 层做了多少**。PM Studio 内置了 harness，龙虾需要用户自己建。对 PM 来说：如果你要用 agent 做工作，先花时间建 harness（规则、检查点、回滚机制），不要直接裸奔。

#openclaw #harness #pm-studio #agent-reliability #method

**发起人**：He Zhang, Jingxia Xing, Dale Xiao | **时间**：10:51 - 11:12
